## structure

*  java
    - 设计模式
    - 反射
    - 线程和线程池
    - Lock 锁
    - 枚举
    - 动态代理
*  android
    - 自定义View
        1. 测量
        2. 布局
        3. 绘制
        4. 仿 Flipboard 的折页效果
        5. 仿即刻的点赞效果
    - touch
        1. 事件分发
        2. NestedScroll
    - AIDL
    - 自定义权限
    - bus
        1. LiveDataBus
        2. RxBus
        3. EventBus
    - 蓝牙
        1. Classic Bluetooth(经典蓝牙)
        2. Bluetooth Low Energy(BLE,低功耗蓝牙)
    - VPN
    - WiFi
*  kotlin
    - 实现单例的方式
